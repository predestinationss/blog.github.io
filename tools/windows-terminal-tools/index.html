Windows Power Shell 也能像 Linux 一样更改主题，持续补充

起因

大部分时候都是用 WSL2 的命令行，但是有的时候不得不用 Windows Power Shell，例如需要使用到USB摄像头时，WSL2 调不到笔记本的摄像头，这个时候就需要切换到 Windows 环境进行验证。另外，Windows 开发环境其实比较劝退人的重要原因之一就是命令行跟 Linux 比起来实在是一言难尽。
