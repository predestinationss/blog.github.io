移动WSL2至其他磁盘

发表于 2022-08-21 分类于 tools

背景

WSL大量占用C盘空间，C盘扩容较繁琐，考虑将WSL移动至其他空间富余磁盘

环境

  • Windows 11 (beta)
  • WSL 2 (Ubuntu 20.04)

工具

move-wsl

方法

  • 运行 Windows PowerShell ,停止WSL服务
  • 拉取 move-wsl 仓库
  • 进入仓库，运行move-wsl.ps1
  • 选择需要移动的发行版
  • 选择Install选项
  • 等待Done
  • 重新启动WSL服务
  • 更改默认用户，移动后默认用户变为root 写入以下内容，YOUR_USERNAME 替换为用户名
  • 重新启动WSL服务
